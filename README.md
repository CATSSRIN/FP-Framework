# üìö Sistem Peminjaman Buku Perpustakaan

Sistem manajemen perpustakaan berbasis web yang dibangun menggunakan Laravel 12 dengan template Mazer. Sistem ini memungkinkan pengelolaan buku, anggota, dan transaksi peminjaman secara efisien.

## üéØ Fitur Utama

- ‚úÖ **CRUD Kategori** - Mengelola kategori buku
- ‚úÖ **CRUD Buku** - Mengelola data buku perpustakaan
- ‚úÖ **CRUD Anggota** - Mengelola data anggota perpustakaan
- ‚úÖ **CRUD Peminjaman** - Mengelola transaksi peminjaman dan pengembalian buku
- ‚úÖ **Role-Based Access Control** - Admin, Pustakawan, dan Anggota
- ‚úÖ **Dashboard** - Statistik dan ringkasan data perpustakaan
- ‚úÖ **Sistem Denda** - Perhitungan denda otomatis untuk keterlambatan

## üõ†Ô∏è Teknologi yang Digunakan

## üíª Instalasi

### Persyaratan Sistem
- PHP >= 8.2
- Composer
- SQLite atau MySQL

### Langkah Instalasi

1. **Clone Repository**
   ```bash
   git clone https://github.com/CATSSRIN/FP-Framework.git
   cd FP-Framework
   ```

2. **Install Dependencies**
   ```bash
   composer install
   ```

3. **Setup Environment**
   ```bash
   cp .env.example .env
   php artisan key:generate
   ```

4. **Buat Database**
   ```bash
   touch database/database.sqlite
   ```

5. **Jalankan Migrasi dan Seeder**
   ```bash
   php artisan migrate --seed
   ```

6. **Jalankan Aplikasi**
   ```bash
   php artisan serve
   ```

7. **Akses Aplikasi**
   Buka browser dan akses: `http://localhost:8000`
